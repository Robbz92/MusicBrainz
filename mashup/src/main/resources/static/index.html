REST-API

http://localhost:8080/MBID/ID

Exempel: http://localhost:8080/MBID/5b11f4ce-a62d-471e-81fc-a69a8278c7da

Mata in MBID ID för artist gruppen. Du kommer få tillbaka ett JSON format i webläsaren.
Det är fördelaktigt ifall du har "JSON formatter" plugin för att kunna läsa datan lättare, alternativt om du använder Postman/Insomnia!
